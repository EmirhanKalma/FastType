##Ключевые возможности

* **Интеллектуальная генерация текстов**: Тексты для тренировок генерируются не по шаблонам, а в реальном времени с помощью локально развернутой нейросети **GPT-2** (Hugging Face / PyTorch).
* **Глубокая аналитика**: Система отслеживает WPM, точность и сохраняет детальную тепловую карту нажатий клавиш пользователя.
* **Кастомизация и Темы**: Более 50 предустановленных цветовых тем, динамические эффекты на Canvas (дождь, сакура, жидкие часы) и полная кастомизация клавиатуры (тени, прозрачность, задержки анимаций).
* **Гибридная архитектура**: Сайт и Telegram-бот подключены к единой базе данных. Любые изменения на сайте мгновенно доступны в боте.
* **Адаптивные алгоритмы**: Алгоритм ручной сортировки (Bubble Sort) для формирования честной таблицы лидеров.
* **AFK Режим**: Заставка «Liquid Clock», плавно появляющаяся при бездействии пользователя.

##Технологический стек

* **Backend**: Python 3, FastAPI, Uvicorn
* **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3, Tailwind CSS, Chart.js
* **AI & ML**: Transformers (Hugging Face), PyTorch
* **Bot**: pyTelegramBotAPI (Telebot)
* **База данных**: Локальный JSON-файл (`data.json`)

После успешного запуска откройте в браузере: http://127.0.0.1:8000
🧠 Интеграция с GPT-2 (Искусственный Интеллект)
Одной из главных архитектурных особенностей FastType является отказ от статичных массивов со словами в пользу динамической генерации.
На бэкенде используется библиотека transformers от Hugging Face:
При старте игры клиент делает запрос к /api/get_random_text.
Сервер случайным образом выбирает "затравку" (prompt), например: "The universe is vast and...".
Локальная модель GPT-2 принимает эту затравку и продолжает предложение, генерируя осмысленный текст с соблюдением синтаксиса и пунктуации.
Текст форматируется (очищается от артефактов и обрывов строк) и отправляется на фронтенд для тренировки.
Это гарантирует, что пользователь каждый раз получает уникальный, живой текст, что повышает качество развития навыка слепой печати.
📱 Использование Telegram-бота
Telegram-бот работает в отдельном демоническом потоке (threading), не блокируя асинхронные веб-запросы FastAPI. Бот обращается к той же базе data.json, что и веб-интерфейс.
Доступные команды:
/start — Информация о проекте и инструкции.
/reg [логин] [пароль] — Регистрация нового аккаунта прямо из мессенджера (после этого можно зайти на веб-сайт под этими же данными).
/sign [логин] [пароль] — Просмотр подробной статистики (WPM, Точность, FP, Активная тема) прямо в чате.
🔒 Скрытая панель Администратора
В проект встроена система управления пользователями. Для доступа к ней реализовано "пасхальное яйцо" (Easter Egg):
Перейдите на любую страницу веб-приложения (например, в Профиль).
В правом нижнем углу найдите текст Version:1.1.0.
Быстро кликните по нему 5 раз подряд.
После неоновой вспышки вы будете перенаправлены в панель управления.
Возможности админки:
Просмотр глобальных метрик (суммарные тесты, ошибки, оверролл FP).
Прямое редактирование любого параметра игрока (начисление очков, изменение рекорда WPM).
Принудительное удаление аккаунта нарушителя/читера из базы данных.
